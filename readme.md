# Brain Tumor Detection using CNN

## Project Description
This project detects the presence of brain tumors from MRI images using a Convolutional Neural Network (CNN) built with TensorFlow and Keras.

## Features
- MRI Image Classification
- Binary Classification (Tumor / No Tumor)
- Data Augmentation
- CNN Model
- Accuracy & Loss Graphs
- Saved Trained Model

## Technologies
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- OpenCV

## Dataset
Brain Tumor MRI Dataset
